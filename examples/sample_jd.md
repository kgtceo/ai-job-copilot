# Senior AI Engineer (Applied LLM)

**About the role**
We're a Series A startup building AI-native products. You'll own LLM features
end to end — from prompt pipelines and retrieval to evaluation and shipping to
production. You'll work directly with the founders and set the bar for how we
build with LLMs.

**What you'll do**
- Design and ship production LLM features: RAG, agents, tool-use, structured
  output pipelines.
- Build evaluation harnesses so we can measure and improve model quality.
- Own the full stack of an AI feature — API, data, and a clean user-facing UI.
- Optimise cost and latency across models; choose the right model per task.

**Requirements (must have)**
- Strong Python. Production experience integrating LLM APIs (OpenAI/Anthropic).
- Experience designing prompt pipelines and handling structured output reliably.
- You've built and shipped real products to real users, not just notebooks.
- Comfort across the stack: you can put a usable UI on top of your models.
- Cloud deployment experience (AWS or similar) and CI/CD.

**Nice to have**
- Experience building evals / LLM-as-judge.
- TypeScript / Next.js.
- Fine-tuning or working with open models.
- Founder or early-startup experience.

**Keywords:** python, llm, anthropic, rag, agents, evals, structured output,
prompt engineering, aws, ci/cd, next.js, fastapi, production
